# Dashboard de Performance Corporativa

Este repositório contém o relatório de Business Intelligence focado no acompanhamento, monitorização e análise dos indicadores de performance corporativa da organização.

## Descrição do Projeto
O Dashboard-performance-corporativa.pbix foi desenvolvido no Microsoft Power BI Desktop para centralizar métricas estratégicas, táticas e operacionais. O objetivo principal é transformar dados brutos em insights visuais de alto impacto para apoiar a tomada de decisão da liderança e gestores.

## Principais Funcionalidades e Visualizações
* **Análise de KPIs Estratégicos:** Monitorização rápida de metas globais da empresa (Realizado vs. Planeado).
* **Filtros Dinâmicos (Segmentação de Dados):** Navegação e filtragem avançada por períodos (Ano, Trimestre, Mês), departamentos ou regiões.
* **Análise de Tendências:** Gráficos evolutivos que ajudam a identificar padrões de desempenho e sazonalidade.
* **Detalhamento (Drill-down):** Capacidade de explorar visões macro até chegar ao nível detalhado das transações.

## Tecnologias Utilizadas
* **Microsoft Power BI Desktop:** Modelagem de dados, relações e desenho dos painéis visuais.
* **Linguagem DAX (Data Analysis Expressions):** Criação de métricas calculadas e lógica de inteligência temporal.
* **Power Query (Linguagem M):** Processo de ETL (Extração, Transformação e Limpeza de Dados).

## Como Executar o Projeto

### Pré-requisitos
É necessário ter o Microsoft Power BI Desktop instalado na sua máquina Windows (disponível gratuitamente na Microsoft Store).

### Passo a Passo
1. Faz o download do ficheiro Dashboard-performance-corporativa.pbix deste repositório.
2. Abre o Power BI Desktop.
3. No menu superior, vai a Ficheiro > Abrir relatório e seleciona o ficheiro descarregado.
4. Nota: Caso as fontes de dados originais sejam locais, pode ser necessário reconfigurar os caminhos em Transformar Dados > Definições da Fonte de Dados.
